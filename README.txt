# Movie Library (Личная кинотека)

## Автор
Максим Филимонов

## Описание программы
**Movie Library** — это десктопное GUI-приложение для управления личной коллекцией фильмов. 
Программа позволяет хранить информацию о названиях, жанрах, годах выпуска и рейтингах фильмов. 
Основные особенности:
- Интуитивно понятный графический интерфейс на базе `tkinter`.
- Валидация вводимых данных (проверка формата года и диапазона рейтинга).
- Динамическая фильтрация списка фильмов по жанру и году.
- Автоматическое сохранение и загрузка базы данных в формате JSON.

## Инструкция по запуску

### Требования
- Установленный интерпретатор **Python 3.6** или выше.
- Наличие библиотеки `tkinter` (входит в стандартную поставку Python на Windows и macOS; на Linux может потребоваться установка пакета `python3-tk`).

### Установка зависимостей
Дополнительные сторонние библиотеки не требуются. Все используемые модули (`json`, `os`, `tkinter`) являются стандартными.
