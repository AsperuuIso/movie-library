# Movie Library (Личная кинотека)

## Автор
Филимонов Максим

## Описание
Графическое приложение на Python (Tkinter) для управления личной коллекцией фильмов. 
Приложение позволяет добавлять фильмы, просматривать их в виде таблицы, а также фильтровать по жанру и году выпуска. 
Данные сохраняются в локальном файле формата JSON.

## Функциональные возможности
- Добавление фильма с указанием названия, жанра, года выпуска и рейтинга.
- Валидация вводимых данных (год — целое число, рейтинг — число от 0 до 10).
- Просмотр списка фильмов в табличном виде.
- Фильтрация списка по жанру и году в реальном времени.
- Автоматическое сохранение и загрузка данных из файла `movies.json`.

## Требования
- Python 3.6+
- Стандартные библиотеки: `tkinter`, `json`, `os`
